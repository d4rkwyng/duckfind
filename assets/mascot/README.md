# DuckFind mascot

Source art for the detective-duck mascot, drawn in Illustrator.

- `duckfind_mascot.ai` — Illustrator source (edit this)
- `duckfind_mascot.png` / `@0.5x` / `@2x` / `@4x` — PNG exports
- `duckfind_mascot.webp` — web export (used on mindtricks.io)
- `duckfind_mascot.gif` — 128x120 transparent GIF, deployed as `public/duck.gif`
